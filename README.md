# 📈 AI Stock Screener

> ML + Deep Learning + Smart Money Concepts for NSE/BSE Indian Stocks

🔴 **Live Demo:** [ai-stock-screener.streamlit.app](https://ai-stock-screener-6.streamlit.app/)

## 🧠 Features
- Fundamental Analysis — 10-point scorecard
- 30+ Technical Indicators
- 13 Candlestick Patterns
- SMC — BOS, CHoCH, Order Blocks, FVG
- ML Signal — Random Forest + XGBoost
- DL Forecast — LSTM 30-day prediction
- Valuation — P/E, PEG, P/B methods

## 🛠️ Tech Stack
Python · Streamlit · TensorFlow · Scikit-learn
XGBoost · yfinance · Plotly · Pandas

## 🚀 Run Locally
pip install -r requirements.txt
streamlit run app.py
